# AS6 Public Brand Master Screen v2

## Diagnostics
The first public brand iteration was clearer but still did not fully inherit the canonical Living Space visual grammar.

## Root Cause
The site was treated as a conventional product landing page instead of the first state of the AS6 Living Space.

## Change
- one child-readable promise: Весь бизнес. В одном месте.;
- one primary CTA: Попробовать бесплатно;
- one canonical Living Space visual in the hero;
- three simple benefits: Собирает, Объясняет, Подсказывает;
- mobile-first hierarchy and reduced-motion support;
- permanent prevention control.
