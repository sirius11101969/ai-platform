# AS6 Living Space Canonical v2

## Diagnostics
Implementation drifted from the approved Master Screen and canonical spatial references toward ordinary dashboard and list layouts.

## Root Cause
Real-data delivery was incorrectly treated as a replacement for the Living Space visual grammar instead of a data source for it.

## Change

- left contextual insight rail;
- central living graph and semantic core;
- right real-time event rail;
- process path, recommendation and intent entry;
- Documents real data rendered through the canonical engine;
- responsive and reduced-motion behavior;
- permanent prevention control.
