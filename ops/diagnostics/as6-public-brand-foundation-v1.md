# AS6 Public Brand Foundation v1

## Diagnostics
The public home explained platform architecture before communicating a one-second human benefit.

## Root Cause
The hero combined multiple abstract promises and competing actions instead of one child-readable value proposition.

## Change
- one-second promise: Ваш бизнес. В одном месте.;
- one primary CTA: Попробовать бесплатно;
- three child-readable steps: Собирает, Объясняет, Подсказывает;
- one continuous visual language from public site to product;
- responsive and reduced-motion controls.
