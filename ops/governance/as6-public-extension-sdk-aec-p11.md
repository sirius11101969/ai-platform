# AS6 Public Extension SDK AEC P11

Failure classes:
- AS6_PUBLIC_EXTENSION_SDK_DRIFT
- AS6_PLUGIN_SDK_COMPATIBILITY_MISMATCH
- AS6_PLUGIN_FACTORY_GAP
- AS6_PLUGIN_DEVELOPER_DOCS_GAP
- AS6_PLUGIN_EXAMPLE_GAP

AEC rules:
- Public plugins should use defineAS6Plugin.
- Public plugins should pass SDK compatibility validation.
- Widget, AI Action and Bus plugins should use SDK factories.
