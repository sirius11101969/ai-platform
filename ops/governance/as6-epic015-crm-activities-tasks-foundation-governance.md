# AS6 Governance — EPIC015 CRM Activities / Tasks Foundation

Stage: AS6_EPIC015_SLICE02_CRM_ACTIVITIES_TASKS_FOUNDATION

- Activities / Tasks foundation must remain declarative.
- Runtime, resolver, navigation and panels must not introduce storage, API calls or business workflow.
- Foundation must reuse existing CRM Workspace and CRM Layout.
- Foundation must not create own router, own store or parallel shell.
- Future UI layer must reuse this foundation descriptor and health snapshot.
