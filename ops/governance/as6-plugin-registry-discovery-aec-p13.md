# AS6 Plugin Registry & Discovery AEC P13

Failure classes:
- AS6_PLUGIN_REGISTRY_DISCOVERY_DRIFT
- AS6_PLUGIN_DISCOVERY_GAP
- AS6_PLUGIN_DUPLICATE_ID
- AS6_PLUGIN_COMPATIBILITY_DISCOVERY_GAP
- AS6_PLUGIN_MARKETPLACE_VISIBILITY_GAP

AEC rules:
- Plugins must be discoverable through Plugin Discovery Engine.
- Duplicate plugin IDs must be detected.
- Plugin registry must expose valid plugin IDs.
- Marketplace UI must consume Plugin Registry state.
