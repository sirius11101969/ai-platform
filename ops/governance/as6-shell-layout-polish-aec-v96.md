# AS6 Shell Layout Polish AEC V96

Failure classes:
- AS6_SHELL_LAYOUT_COMPOSITION_DRIFT
- AS6_SHELL_SURFACE_OVERLAP_RISK
- AS6_PAGE_LEVEL_SHELL_LAYOUT_DUPLICATION
- AS6_SHELL_RESPONSIVE_BREAKPOINT_GAP

AEC rules:
- Shell surfaces must be composed in AS6Shell, not page components.
- Nav and Context Bar belong to the top shell composition zone.
- Intelligence Rail belongs to the rail composition zone.
- Shell layout must include tablet and mobile breakpoints.
