# AS6 OS Assistant Command Layer Governance

- Rule: AS6 Assistant must be visible as a persistent command affordance where it improves usability.
- Rule: Command layer must not block CRM cards, drag-and-drop, forms or mobile usability.
- Rule: Command layer changes must remain non-destructive CSS until interactive command palette is implemented.
- AEC rule: assistant-command-layer-requires-build-import-count-and-mobile-safety.
- Failure class: assistant-command-affordance-gap.
