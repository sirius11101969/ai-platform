# AS6 Governance — EPIC015 CRM Activities / Tasks CRM Layout Bridge

Stage: AS6_EPIC015_SLICE05_CRM_ACTIVITIES_TASKS_CRM_LAYOUT_BRIDGE

- Activities / Tasks CRM Layout Bridge must reuse existing CRM Layout.
- Activities / Tasks CRM Layout Bridge must reuse existing CRM Workspace, Header and Sidebar.
- Layout bridge must not introduce own router, own store, storage, API calls or business workflow.
- Breadcrumbs and active section must be declarative.
- Future Live Layout Mount must reuse this layout descriptor and health snapshot.
