# AS6 Governance — EPIC014 CRM Deals Workspace Integration

Stage: AS6_EPIC014_SLICE04_CRM_DEALS_WORKSPACE_INTEGRATION

- Deals must integrate into the existing CRM Workspace.
- Deals must reuse DealsUiFoundation as the workspace panel component.
- Deals must reuse existing CRM Layout and must not create a parallel shell or layout.
- Deals must not create an own router or store.
- Storage, API calls and business workflow remain forbidden.
- Future CRM Layout Bridge must reuse this workspace integration.
