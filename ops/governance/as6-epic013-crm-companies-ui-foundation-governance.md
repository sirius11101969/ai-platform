# AS6 Governance — EPIC013 CRM Companies UI Foundation

Stage: AS6_EPIC013_SLICE03_CRM_COMPANIES_UI_FOUNDATION

- Companies UI Foundation must remain declarative and reusable.
- UI states loading, empty, ready and error are required.
- Storage, API calls, backend coupling and business workflows are forbidden.
- UI must expose diagnostic selectors for list, card, states and diagnostics panel.
- Future Workspace Integration must reuse this UI foundation without creating a parallel shell.
