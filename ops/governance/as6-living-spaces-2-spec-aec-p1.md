# AS6 Living Spaces 2.0 Specification AEC P1

Failure classes:
- AS6_LIVING_SPACE_2_SPEC_DRIFT
- AS6_SPACE_MANIFEST_SCHEMA_GAP
- AS6_SPACE_RUNTIME_GAP
- AS6_SPACE_CONTEXT_GAP
- AS6_SPACE_REFERENCE_IMPLEMENTATION_GAP

AEC rules:
- New Living Spaces must provide a manifest.
- Space manifests must validate against as6SpaceManifest.schema.js.
- Space lifecycle must use AS6 Space Runtime.
- Space context must use AS6 Space Context model.
- CRM is the first reference manifest for Living Spaces 2.0.
