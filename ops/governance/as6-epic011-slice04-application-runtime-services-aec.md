# AS6 EPIC011 Slice 04 AEC

- Runtime Services must depend on contracts.
- Runtime Services must communicate through Event Bus and Runtime Context.
- Runtime Services must not depend on each other directly.
- Require Runtime Service Registry.
- Require Runtime Service Contract.
- Require Service Resolver.
- Require Dependency Resolver.
- Require Service Container.
- Require Runtime Context Bridge.
- Prevent business logic and CRM logic in Runtime Services.
- Prevent platform baseline mutation.
