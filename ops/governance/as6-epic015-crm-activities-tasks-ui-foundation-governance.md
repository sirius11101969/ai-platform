# AS6 Governance — EPIC015 CRM Activities / Tasks UI Foundation

Stage: AS6_EPIC015_SLICE03_CRM_ACTIVITIES_TASKS_UI_FOUNDATION

- Activities / Tasks UI Foundation must remain presentation-only.
- UI placeholders must not introduce storage, API calls or business workflow.
- UI must reuse Activities / Tasks foundation health snapshot.
- UI must not create own router, own store or parallel shell.
- Future Workspace Integration must reuse ActivitiesUiFoundation.
