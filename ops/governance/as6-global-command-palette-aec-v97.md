# AS6 Global Command Palette AEC V97

Failure classes:
- AS6_GLOBAL_COMMAND_PALETTE_DRIFT
- AS6_COMMAND_PALETTE_ENGINE_BYPASS
- AS6_KEYBOARD_NAVIGATION_GAP
- AS6_COMMAND_ROUTE_DUPLICATION_RISK

AEC rules:
- Global command palette must use getAS6LivingSpaceMenuItems().
- Ctrl+K must open the shell-level command surface.
- Navigation commands must use router navigation.
- Command Palette must not duplicate Living Space registry data.
