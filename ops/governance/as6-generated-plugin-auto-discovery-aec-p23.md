# AS6 Generated Plugin Auto Discovery AEC P23

Failure classes:
- AS6_GENERATED_PLUGIN_AUTO_DISCOVERY_DRIFT
- AS6_GENERATED_PLUGIN_REGISTRY_GAP
- AS6_GENERATED_PLUGIN_MANIFEST_INDEX_GAP
- AS6_PLUGIN_REGISTRY_GENERATED_SOURCE_GAP

AEC rules:
- Generated plugins must be exported from generated/index.js.
- Generated plugin manifest must list generated plugins.
- AS6PluginRegistry must include getAS6GeneratedPlugins().
