# AS6 Marketplace UI / Developer Console AEC P14

Failure classes:
- AS6_MARKETPLACE_UI_DEVELOPER_CONSOLE_DRIFT
- AS6_PLUGIN_REGISTRY_UI_GAP
- AS6_PLUGIN_COMPATIBILITY_VISIBILITY_GAP
- AS6_PLUGIN_LIFECYCLE_ACTION_UI_GAP
- AS6_DEVELOPER_CONSOLE_GAP

AEC rules:
- Marketplace UI must display Plugin Registry state.
- Marketplace UI must display SDK compatibility.
- Developer Console must expose enable/disable/remove actions.
- Marketplace UI must remain backed by Plugin Registry and Plugin Runtime.
