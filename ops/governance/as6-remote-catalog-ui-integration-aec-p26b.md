# AS6 Remote Catalog UI Integration AEC P26B

Failure classes:
- AS6_REMOTE_CATALOG_UI_INTEGRATION_DRIFT
- AS6_MARKETPLACE_REMOTE_SYNC_UI_GAP
- AS6_REMOTE_CATALOG_STATUS_VISIBILITY_GAP

AEC rules:
- Developer Console must expose remote catalog source.
- Developer Console must expose manual sync control.
- Remote catalog errors must be visible without breaking local fallback.
