# AS6 CRM Contacts CRM Layout Bridge Governance

Stage: AS6_EPIC012_SLICE07_CRM_CONTACTS_CRM_LAYOUT_BRIDGE

- Contacts must bridge into the unified CRM layout.
- Contacts must reuse the existing workspace panel and navigation.
- Contacts must not create isolated layout containers.
- Breadcrumbs and active section must be synchronized.
- Layout states are limited to loading, empty, ready and error.
- Storage, API calls and business workflows remain forbidden in this slice.
