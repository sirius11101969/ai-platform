# AS6 Living Spaces 2.0 Specification Root Cause P1

Root cause: AS6 had Living Spaces as routes and shell modules, but no complete product-level contract for spaces as full platform applications.

Risk: future spaces such as CRM, Sales, Finance, Tasks, Files and AI could drift into incompatible implementations.

Repair: add Living Spaces 2.0 specification, manifest schema, runtime, context model, registry and CRM reference manifest.
