# AS6 Governance — EPIC013 CRM Companies Live Layout Mount

Stage: AS6_EPIC013_SLICE06_CRM_COMPANIES_LIVE_LAYOUT_MOUNT

- Companies must be mounted in the existing CRMPage production layout.
- Companies must reuse CRM Layout Bridge, Layout Snapshot and UI Foundation.
- Companies must not create a parallel shell, router, store or layout.
- Storage, API calls, backend coupling and business workflows remain forbidden.
- Diagnostics must preserve CRMPage before/diff evidence.
