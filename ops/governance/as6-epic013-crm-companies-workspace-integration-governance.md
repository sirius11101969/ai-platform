# AS6 Governance — EPIC013 CRM Companies Workspace Integration

Stage: AS6_EPIC013_SLICE04_CRM_COMPANIES_WORKSPACE_INTEGRATION

- Companies must integrate through the existing CRM Workspace pattern.
- Companies must register a workspace panel and navigation declaratively.
- Companies must not create a parallel shell.
- Companies must preserve loading, empty, ready and error states.
- Storage, API calls, backend coupling and business workflows remain forbidden.
- Future CRM Layout Bridge must reuse this Workspace Integration.
