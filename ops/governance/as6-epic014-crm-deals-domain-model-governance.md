# AS6 Governance — EPIC014 CRM Deals Domain Model

Stage: AS6_EPIC014_SLICE01_CRM_DEALS_DOMAIN_MODEL

- Deals domain model must remain declarative.
- Deals must reuse Contacts and Companies foundations without hard coupling.
- Deals initial slices must not introduce storage, API calls or business workflows.
- Pipeline, stages and lifecycle are domain declarations only at Slice 01.
- Future foundation/UI/workspace slices must reuse this domain contract.
