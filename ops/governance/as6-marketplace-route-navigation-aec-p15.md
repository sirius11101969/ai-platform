# AS6 Marketplace Route & Navigation AEC P15

Failure classes:
- AS6_MARKETPLACE_ROUTE_NAVIGATION_DRIFT
- AS6_MARKETPLACE_ROUTE_GAP
- AS6_MARKETPLACE_NAVIGATION_GAP
- AS6_MARKETPLACE_CONSOLE_ROUTE_BYPASS
- AS6_DYNAMIC_NAVIGATION_METADATA_GAP

AEC rules:
- Marketplace Console must have a route-ready page.
- Marketplace route must expose /marketplace path metadata.
- Marketplace navigation must be exported from plugin route module.
- AS6 Shell integration must consume route/navigation metadata in next stage.
