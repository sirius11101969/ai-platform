# AS6 Autonomous Change Controller

The Autonomous Change Controller is the production gate that prevents unsafe autonomous or manual changes.

Controller requirements:
- AS6_AUTONOMOUS_CHANGE_IMPACT=PASS
- AS6_AUTONOMOUS_CHANGE_RISK=PASS
- AS6_AUTONOMOUS_CHANGE_ROLLBACK=PASS
- AS6_AUTONOMOUS_CHANGE_VALIDATION=PASS
- AS6_AUTONOMOUS_CHANGE_EVIDENCE=PASS
- AS6_AUTONOMOUS_CHANGE_REGISTRY=PASS
- AS6_AUTONOMOUS_CHANGE_DEPLOYMENT_GATE=PASS

Blocking root cause classes:
- AUTONOMOUS_CHANGE_CONTROLLER_DRIFT
- CHANGE_EVIDENCE_MISSING
- DEPLOYMENT_GATE_BYPASS
- UNVALIDATED_AUTONOMOUS_CHANGE

AEC signal required by diagnostics:
AEC_AUTONOMOUS_CHANGE_CONTROLLER=PASS
