# AS6 Governance — EPIC015 CRM Activities / Tasks Workspace Integration

Stage: AS6_EPIC015_SLICE04_CRM_ACTIVITIES_TASKS_WORKSPACE_INTEGRATION

- Activities / Tasks Workspace Integration must reuse existing CRM Workspace.
- Activities / Tasks Workspace Integration must reuse existing CRM Layout.
