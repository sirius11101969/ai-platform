# AS6 Release Evidence Manifest AEC V118

Failure classes:
- AS6_RELEASE_EVIDENCE_MANIFEST_DRIFT
- AS6_RELEASE_EVIDENCE_MISSING
- AS6_RELEASE_EVIDENCE_TARGET_GAP
- AS6_RELEASE_EVIDENCE_READINESS_GAP

AEC rules:
- Release cycles must produce evidence manifest.
- Evidence must include branch, head, target, validate, release gate and readiness.
- Release evidence must be stored under ops/evidence.
