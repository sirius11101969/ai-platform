# AS6 Capability Registry AEC V113

Failure classes:
- AS6_CAPABILITY_REGISTRY_DRIFT
- AS6_CAPABILITY_METADATA_DUPLICATION_RISK
- AS6_CAPABILITY_STATUS_GOVERNANCE_GAP
- AS6_CAPABILITY_PRIORITY_DRIFT
- AS6_CAPABILITY_SERVICE_BINDING_GAP

AEC rules:
- Capabilities must be registered in as6CapabilityRegistry.js.
- Capability routing should use Capability Engine.
- Capability status and priority must be governed centrally.
- Capability service/plugin bindings must be explicit.
