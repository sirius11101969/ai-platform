# AS6 Governance — EPIC014 CRM Deals Live Layout Mount

Stage: AS6_EPIC014_SLICE06_CRM_DEALS_LIVE_LAYOUT_MOUNT

- Deals must mount only through the existing CRMPage production layout.
- Deals must reuse existing CRM Layout and Workspace.
- Duplicate route and duplicate panel registration are forbidden.
- Own router, own store, parallel layout and parallel shell are forbidden.
- Storage, API calls, backend coupling and business workflows remain forbidden.
