# AS6 Governance — EPIC014 CRM Deals UI Foundation

Stage: AS6_EPIC014_SLICE03_CRM_DEALS_UI_FOUNDATION

- Deals UI Foundation must remain declarative and foundation-only.
- Deals UI must expose loading, empty, ready and error states.
- Deals UI must reuse Deals Foundation health snapshot and panels.
- Deals UI must not introduce storage, API calls, backend coupling or business workflow.
- Deals UI must not introduce heavy dependencies.
- Future workspace integration must reuse DealsUiFoundation.
