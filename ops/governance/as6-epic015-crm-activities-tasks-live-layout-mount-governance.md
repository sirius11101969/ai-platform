# AS6 Governance — EPIC015 CRM Activities / Tasks Live Layout Mount

Stage: AS6_EPIC015_SLICE06_CRM_ACTIVITIES_TASKS_LIVE_LAYOUT_MOUNT

- Activities / Tasks Live Layout Mount must be mounted through existing CRMPage.
- Live mount must reuse existing CRM Layout, Workspace, Header, Sidebar and router.
- Live mount must not create own router, own store, duplicate route, duplicate panel, storage, API calls or business workflow.
- Future Production Polish must reuse this live mount health snapshot and diagnostics.
