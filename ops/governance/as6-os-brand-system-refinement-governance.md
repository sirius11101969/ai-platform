# AS6 OS Brand System Refinement Governance

- Rule: AS6 должен восприниматься как AI Operating System, а CRM — как модуль внутри OS.
- Rule: brand polish не должен ломать route, auth, CRM API, lead data, drag-and-drop и production flow.
- Control: только CSS layer + один безопасный import.
- Failure class: crm-saas-brand-drift.
- AEC rule: любые OS/CRM visual changes должны проходить build, import-count check, secret scan и registry update.
