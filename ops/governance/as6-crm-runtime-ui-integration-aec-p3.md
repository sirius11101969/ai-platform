# AS6 CRM Runtime UI Integration AEC P3

Failure classes:
- AS6_CRM_RUNTIME_UI_INTEGRATION_DRIFT
- AS6_CRM_RUNTIME_ACTIVATION_GAP
- AS6_CRM_RUNTIME_CONTEXT_VISIBILITY_GAP
- AS6_CRM_SHELL_RUNTIME_BYPASS

AEC rules:
- CRM/Sales shell must activate CRM Living Space Runtime.
- CRM/Sales shell must expose runtime status marker.
- CRM runtime context must be updateable from UI bridge.
- CRM UI integration must not bypass Space Runtime.
