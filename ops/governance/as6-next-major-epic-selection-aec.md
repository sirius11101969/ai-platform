# AS6 Next Major EPIC Selection AEC

- NEXT_MAJOR_EPIC_SELECTION selects direction only.
- NEXT_MAJOR_EPIC_SELECTION must not authorize implementation.
- Selected EPIC must declare OWNING_BASELINE.
- Selected EPIC must declare AFFECTED_COMPONENTS.
- Selected EPIC must reference required planning artifacts.
- Application EPICs must build on AS6_OPERATING_SYSTEM_V1.
- Platform baseline changes require separate engineering decision, EPIC and ADR.
