# AS6 Remote Marketplace Catalog AEC P26A

Failure classes:
- AS6_REMOTE_MARKETPLACE_CATALOG_DRIFT
- AS6_MARKETPLACE_REMOTE_CATALOG_GAP
- AS6_MARKETPLACE_CATALOG_FALLBACK_GAP
- AS6_REMOTE_PLUGIN_SYNC_CONTRACT_GAP

AEC rules:
- Remote Marketplace Catalog must provide local fallback.
- Remote catalog plugins must be normalizable before registration.
- Remote catalog must not break local Marketplace behavior.
