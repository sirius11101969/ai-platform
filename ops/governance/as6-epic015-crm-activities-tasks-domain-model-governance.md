# AS6 Governance — EPIC015 CRM Activities / Tasks Domain Model

Stage: AS6_EPIC015_SLICE01_CRM_ACTIVITIES_TASKS_DOMAIN_MODEL

- Activities / Tasks domain model must remain declarative.
- Links to Companies, Contacts and Deals must remain declarative-only.
- Deadline and reminder models must not introduce storage, API calls or automation workflow in Slice 01.
- Initial Activities / Tasks slices must not mutate platform architecture.
- Future slices must reuse existing CRM Workspace and CRM Layout.
