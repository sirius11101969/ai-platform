# AS6 Governance — EPIC014 CRM Deals Production Polish

Stage: AS6_EPIC014_SLICE07_CRM_DEALS_PRODUCTION_POLISH

- Deals production layer must preserve existing CRM Layout and Workspace.
- Deals production UI must include accessibility and visible focus safeguards.
- Deals production UI must support prefers-reduced-motion.
