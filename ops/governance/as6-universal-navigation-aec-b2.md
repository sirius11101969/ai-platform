# AS6 Universal Navigation AEC B2

Failure classes:
- AS6_BUSINESS_UNIVERSAL_NAVIGATION_DRIFT
- AS6_BUSINESS_NAVIGATION_REGISTRY_GAP
- AS6_BUSINESS_NAVIGATION_TREE_GAP
- AS6_BUSINESS_NAVIGATION_SEARCH_GAP
- AS6_BUSINESS_MARKETPLACE_NAVIGATION_BRIDGE_GAP

AEC rules:
- Business modules must register navigation through Universal Navigation API.
- Existing App/Shell/Sidebar screens must not be replaced during B2.
- Universal Navigation must expose section, item, tree, search, favorite and open primitives.
