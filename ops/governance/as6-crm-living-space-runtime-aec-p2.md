# AS6 CRM Living Space Runtime AEC P2

Failure classes:
- AS6_CRM_LIVING_SPACE_RUNTIME_DRIFT
- AS6_CRM_SPACE_MANIFEST_INVALID
- AS6_CRM_SPACE_CONTEXT_GAP
- AS6_CRM_SPACE_RUNTIME_BYPASS
- AS6_CRM_SPACE_REFERENCE_IMPLEMENTATION_GAP

AEC rules:
- CRM must use crmLivingSpaceManifest as its Platform V2 manifest.
- CRM runtime must register through AS6 Space Registry.
- CRM activation must use AS6 Space Runtime.
- CRM context must use AS6 Space Context.
- CRM is the first reference implementation for Living Spaces 2.0.
