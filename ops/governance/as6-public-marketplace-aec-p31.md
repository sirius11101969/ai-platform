# AS6 Public Marketplace AEC P31

Failure classes:
- AS6_PUBLIC_MARKETPLACE_DRIFT
- AS6_PUBLIC_MARKETPLACE_SOURCE_GAP
- AS6_PUBLIC_MARKETPLACE_SEARCH_GAP
- AS6_PUBLIC_MARKETPLACE_CATEGORY_GAP
- AS6_PUBLIC_MARKETPLACE_TRUST_SUMMARY_GAP

AEC rules:
- Public Marketplace must expose sources.
- Public Marketplace must expose categories.
- Public Marketplace must support search and filters.
- Public Marketplace listings must include trust summary.
