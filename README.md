# AS6

## Спецификация изменения рабочего пространства

### Изменения

- Уменьшить название рабочего пространства `AS6` на 50%.
- Увеличить стрелку раскрытия рядом с названием примерно на 30%.
- Не менять положение блока, логотип и остальную композицию.
- Сохранить существующие шрифт, цвет и стиль.

### Ожидаемый результат

- Логотип остаётся главным акцентом в левом верхнем углу.
- Надпись `AS6` становится компактным служебным элементом.
- Стрелка лучше считывается как элемент раскрытия и становится удобнее для восприятия.

### Критерии приёмки

- Размер надписи `AS6` составляет 50% от текущего размера.
- Размер стрелки раскрытия составляет примерно 130% от текущего размера.
- Положение блока, размеры и положение логотипа не изменены.
- Композиция, шрифт, цвет и визуальный стиль сохранены.
