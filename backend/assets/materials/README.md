# Telegram materials placeholders

Place real Telegram-sendable sales assets in this directory with these exact names:

- `presentation.pdf` — sent via Telegram `sendDocument`
- `demo.mp4` — sent via Telegram `sendVideo`
- `screenshot-1.png` — sent via Telegram `sendPhoto`
- `screenshot-2.png` — sent via Telegram `sendPhoto`
