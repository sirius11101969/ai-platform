# EPIC-007 PR3 — AI Workspace Governance

- AI Workspace must be embedded into Workspace, not implemented as a parallel module shell.
- AI features must use Workspace Context.
- AI Actions must register through Workspace Actions.
- Recommendations must be derived from active module, focus context, right rail, actions and events.
- No separate AI Context is allowed without a new ADR.
- No new AI persistent storage is allowed without explicit approval.
- QGT-007 Context Awareness is mandatory for AI Workspace.
