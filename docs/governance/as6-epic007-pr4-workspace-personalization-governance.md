# EPIC-007 PR4 — Workspace Personalization Governance

- Workspace Personalization must personalize the existing Workspace, not create a new shell.
- Role-based Workspace must reuse existing Workspace Context.
- Personalized AI must use Workspace Profile, User Role, Active Module and Focus Context.
- No separate personalization runtime is allowed without ADR.
- No new persistent storage is allowed without explicit approval.
- INV-009 Single AI Workspace is mandatory.
