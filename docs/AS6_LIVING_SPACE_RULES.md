# AS6 Living Space Rules

Master Screen: Screen 1 is the visual reference.

Rules:
- All screens are states of one Living Space.
- Screen 1: Focus — Пространство концентрации.
- Screen 2: CRM — Пространство отношений.
- Screen 3: Финансы — Пространство устойчивости.
- Документы — Пространство знаний.
- Проекты — Пространство развития.
- Команда — Пространство взаимодействия.
- Решения — Пространство выбора.
- Автоматизация — Пространство автономности.
- Пульс бизнеса — Пространство состояния.
- Совместное мышление — Пространство синхронизации.
- Присутствие — Пространство сопровождения.
- Знания — Пространство памяти.

UI governance:
- Never create a separate interface for each screen.
- Preserve the Master Screen composition, atmosphere, typography and visual hierarchy.
- Percent indicators must match the Master Screen typography and visual weight.
- Do not add decorative frames, dots or unnecessary contours.
- Changes must be evolutionary and preserve AS6 Living Sky.
