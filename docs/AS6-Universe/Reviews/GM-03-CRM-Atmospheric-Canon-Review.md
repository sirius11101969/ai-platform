# GM-03 CRM Atmospheric Canon Review

## Status

PASS / APPROVED / FROZEN

## Diagnostic Summary

### Detected visual failure classes

1. `AS6_VISUAL_ATTENTION_COMPETITION`
   - Multiple headings and labels competed with the central living process.

2. `AS6_CONTAINER_OVEREXPOSURE`
   - Glass frames and panels became visible as interface furniture.

3. `AS6_TYPOGRAPHY_OVERASSERTION`
   - Excessive text weight reduced calmness and distracted from the central scene.

4. `AS6_TEXT_GLOW_ROLE_COLLISION`
   - Explanatory text behaved like an active luminous process.

5. `AS6_NAVIGATION_CONTAINER_DOMINANCE`
   - The top space capsule competed with the Living Space.

6. `AS6_ACTION_ZONE_DISSOLUTION_GAP`
   - Recommendation and input zones needed to remain understandable after container removal.

7. `AS6_CROSS_SPACE_ATMOSPHERIC_DRIFT`
   - Focus and Finance do not yet inherit the approved calm atmospheric layer.

## Root Cause

The previous design language treated glass, frames and strong typography as premium signals. After the central living scenes matured, those same devices became competing visual objects and created attention fragmentation.

## Structural Decision

Separate the AS6 design system into two inheritance layers:

### Structural Canon

Inherited from GM-01 Focus:

- screen architecture;
- navigation roles;
- right-side intelligence area;
- recommendation area;
- conversation area;
- one primary visual center.

### Atmospheric Canon

Inherited from GM-03 CRM:

- calm attention;
- invisible interface;
- progressive visibility;
- interface dissolution;
- semantic light;
- matte non-competing typography;
- containers only when interaction clarity requires them.

## Changes Added to Diagnostics

The following checks are now required for every Living Space review:

- detect more than one competing primary text anchor;
- detect persistent bright borders around passive elements;
- detect white or luminous explanatory text over bright scenes;
- detect navigation containers stronger than the Living Space;
- detect recommendation containers that visually separate the atmosphere;
- detect input fields that dominate in their idle state;
- detect atmospheric drift between approved Golden Masters.

## Preventive Controls

1. `Calm Attention Test`
   - View for 10 seconds without reading.
   - The central process must remain the only dominant impression.

2. `Container Removal Test`
   - Temporarily remove every frame.
   - Restore only those required for interaction clarity.

3. `Semantic Light Test`
   - Light must identify life, activity or change.
   - Explanatory text must remain matte.

4. `Cross-Space Feeling Test`
   - Compare Focus, Finance and CRM side by side.
   - Atmosphere must feel related while geometry remains unique.

5. `Eight-Hour Calmness Test`
   - No secondary heading may demand repeated attention.

## AEC Rules

- AEC-CALM-001: reject any passive frame that is visually stronger than its content.
- AEC-CALM-002: reject equal high contrast across all text levels.
- AEC-CALM-003: reject explanatory text with active glow.
- AEC-CALM-004: reject cross-space atmospheric inconsistency.
- AEC-CALM-005: reject a Living Space if the central process is not the first visual impression.
- AEC-CALM-006: reject an idle input field that appears as a dominant container.
