# AS6 Typography Standard

## Status

APPROVED

## Purpose

Typography is part of the AS6 brand identity and must remain constant across all Living Spaces.

A user should feel that the Living Space changed, not that the product design changed.

## Core Rule

All AS6 Living Spaces use the same calm, premium, blue-gray typography language.

Typography must never become saturated, toxic, bright or visually aggressive.

## Constant Across All Spaces

- font family;
- font sizes;
- line heights;
- letter spacing;
- title style;
- object label style;
- right panel text style;
- bottom card text style;
- input placeholder style;
- base blue-gray color tone.

## Forbidden

- toxic saturated blue text;
- pure black text for primary AS6 UI;
- random color changes per space;
- using accent colors as the main text color;
- making typography compete with the central composition.

## Approved Direction

The approved Focus Golden Master corrected the text color to a calm blue-gray tone.

This typography direction is now the standard for all remaining spaces:

- Финансовая устойчивость;
- Пространство отношений;
- Пространство проектов;
- Пространство документов;
- Пространство знаний;
- Пространство команды;
- Пространство решений;
- Совместное мышление;
- Пространство автоматизации;
- Пульс бизнеса;
- Пространство присутствия.

## Product Principle

The user should notice the identity of each Living Space through its central composition, objects, geometry, material and accent light — not through changing typography.
