## Canonical Rules

- One universe: AS6.
- No classic pages.
- No classic menu as primary navigation.
- Living Spaces are first-level architecture.
- Living Objects belong to the currently active Living Space.
- The base atmosphere is constant across all spaces.
- The central composition creates the identity of each space.
- Typography remains constant across all spaces.
- Golden Master 01 Focus is the first approved visual standard.
