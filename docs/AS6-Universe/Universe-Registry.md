## Status Rules

- DRAFT: concept exists, not visually approved.
- REVIEW: visual direction exists, needs corrections.
- FINAL REVIEW: almost approved, only final standard alignment remains.
- GOLDEN MASTER APPROVED: accepted visual standard.
- FROZEN: no further changes except global AS6 design-language changes.
