# AS6 Platform V2 — Marketplace UI / Developer Console P14

## Purpose

P14 adds a UI-level Developer Console for plugin registry and marketplace management.

## Features

- plugin registry list
- plugin search
- compatibility status
- enable plugin
- disable plugin
- remove plugin
- last action output

## Data sources

- Plugin Registry Discovery P13
- Plugin Runtime P10
- Plugin Marketplace P10
- Public Extension SDK P11

## Next target

Marketplace route integration into AS6 Shell.
