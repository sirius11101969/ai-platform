# AS6 Platform V2 — Public Extension SDK P11

Public SDK for AS6 plugin developers.

## SDK exports

- defineAS6Plugin
- validateAS6PluginSDKCompatibility
- createAS6PluginTemplate
- createAS6WidgetPlugin
- createAS6AIActionPlugin
- createAS6BusPlugin

## Supported extension points

- Living Spaces
- Widgets
- AI Actions
- Universal Service Bus handlers
- Permissions
- Capabilities

## Compatibility

- AS6_PLUGIN_SDK_VERSION=P11
- AS6_PLUGIN_SDK_CORE_COMPATIBILITY=AS6_PLATFORM_V2

## Minimal plugin

Use defineAS6Plugin and pass SDK compatibility validation before publishing.
