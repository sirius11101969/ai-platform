# AS6 Workspace Guidelines

DOCUMENT_TYPE=WORKSPACE_GUIDELINES
DOCUMENT_STATUS=IMPLEMENTATION_GUIDANCE

## Rules

- Every user-facing module must integrate into the unified Workspace.
- Do not create module-specific shells when Workspace Shell exists.
- Reuse Design System components before creating new UI components.
- Use Workspace Context before creating local module context.
- AI features must appear as one AS6 AI layer, not separate AI islands.
