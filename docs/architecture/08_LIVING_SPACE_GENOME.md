# AS6 Living Space Genome v1.0

## Purpose

AS6 is not a set of screens.

AS6 is one continuous Living Space where every product area is a state of the same environment.

The primary rule is:

**Brand is constant. State is variable.**

## Master Screen

Master Screen 3 is the visual and emotional reference for the entire product.

All future states must inherit:

- atmosphere;
- background;
- light;
- typography;
- panels;
- materials;
- motion behavior;
- emotional calm;
- spatial continuity.

## Brand DNA

The following layers are immutable across all states:

- soft volumetric light;
- calm neutral base;
- glass-like and soft surfaces;
- low-noise panels;
- readable typography;
- calm motion;
- depth-based hierarchy;
- single-space continuity.

## Variable Layer

Only the central geometry changes.

Each state receives its own unique central composition and emotional meaning.

No state may reuse the central geometry of another state.

## Core States

- Living Space: calm sphere, home, safety.
- Focus: one center, concentration.
- Thinking: asymmetric gravitational composition, birth of thought.
- Living Ecosystem: living network, business organism.
- Analytics: data crystal, clarity.
- Decision: convergence point, confidence.
- Diagnostics: scanning layers, transparency.
- Knowledge: constellation, context.
- Automation: flowing streams, ease.
- Strategy: vector, direction.
- Evolution: unfolding spiral, growth.

## New State Gate

A new AS6 state is approved only when it answers:

1. What task does it solve?
2. What emotion does it create?
3. What unique central geometry expresses it?
4. How is it different from all existing states?
5. Why does the user recognize AS6 in less than one second?

## Prohibited Drift

- separate page style;
- repeated central geometry;
- broken Master Screen atmosphere;
- decorative motion without meaning;
- competing visual centers;
- interface fragments not born from the space.

## Result

The Genome is the source rule for all future interface work in AS6.
