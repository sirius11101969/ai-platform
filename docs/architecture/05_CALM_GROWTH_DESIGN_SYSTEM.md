# Calm Growth Design System

Calm Growth Design System defines the visual behavior of AS6.

The interface must stay calm while the business space becomes smarter.

## Rules

- Keep the light blue atmosphere.
- Keep high readability.
- Avoid visual noise.
- Use soft motion instead of aggressive alerts.
- Prefer living transitions over page jumps.
- Make every change feel natural.

## Principle

More intelligence must create more calm, not more complexity.
