# AS6 Geometry Atlas v1.0

## Purpose

AS6 Geometry Atlas defines the central geometry of every Living Space state.

A state is not a page. A state is a unique spatial condition inside one AS6 environment.

Primary rule:

**One geometry = one state = one emotion.**

## State Map

- Living Space: master home state.
- Focus: maximum concentration.
- Thinking: idea formation.
- Living Ecosystem: connected business organism.
- Diagnostics: root-cause inspection.
- Analytics: information-to-clarity conversion.
- Decision: choice and commitment.
- Strategy: direction and checkpoints.
- Knowledge: connected memory and context.
- Automation: autonomous process flow.
- Collaboration: multiple centers forming trust.
- Planning: time arcs and priority structure.
- Evolution: unfolding growth spiral.

## Transformation Rules

State change is never a page switch.

The central geometry must transform smoothly and preserve:

- light;
- atmosphere;
- scale;
- depth;
- material language;
- spatial rhythm;
- AS6 brand recognition.

## Approval Gate

Before adding a new state, confirm:

- unique geometry;
- unique emotion;
- no composition duplication;
- Master Screen inheritance;
- one-second AS6 recognition;
- smooth transformation path from another state.
