# AS6 Product Alignment

Every product change in AS6 must answer five questions before implementation:

1. Does it help the user understand the business faster?
2. Does it help the user make a better decision?
3. Does it reduce confusion instead of adding noise?
4. Does it preserve trust and respect for the human?
5. Does it match Canon 1.0 and the AS6 North Star?

If the answer is not clear, the change must be simplified or rejected.

AS6 does not add features only because they are technically possible.
AS6 adds only what helps a person understand, decide and move forward with confidence.
