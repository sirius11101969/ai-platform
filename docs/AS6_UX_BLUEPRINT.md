# AS6 UX BLUEPRINT

## Mission
AS6 — Ваш бизнес. Простыми словами.

## Product Promise
AS6 помогает принимать умные решения.

## Product Identity
AS6 — личный центр управления компанией, который помогает принимать умные решения, видеть возможности для роста и уверенно двигаться вперёд.

## Emotional KPI
После входа в AS6 пользователь должен чувствовать:

- Я понимаю ситуацию.
- Я знаю следующий шаг.
- У меня есть план.
- У меня есть решение.
- Сегодня будет хороший день.

## What AS6 Sells
AS6 продаёт не CRM, AI, отчёты или автоматизацию.

AS6 продаёт:

- уверенность;
- понимание;
- контроль;
- план;
- рост.

## Command Center First Screen
Первый экран Command Center должен быть понятен за 30 секунд и показывать:

- Сегодня заработано;
- процент роста крупнее суммы;
- сумму в рублях основной валютой;
- долларовую сумму как дополнительную справку;
- настроение бизнеса;
- утренний брифинг;
- блок «С чего начать?»;
- подпись «Что лучше сделать сейчас»;
- блок «Спросить AS6»;
- здоровье бизнеса;
- кнопку «Объяснить простыми словами»;
- кнопку «Покажи главное».

Не писать: Продажи упали на 7%.

## Required User Feeling
Через 30 секунд пользователь должен понимать:

- что происходит;
- что важно;
- что лучше сделать сейчас;
- какое есть решение;
- какой следующий шаг.
