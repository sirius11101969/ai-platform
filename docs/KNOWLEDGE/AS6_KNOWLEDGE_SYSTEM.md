# AS6 Knowledge System

AS6 хранит не только код, но и проверяемую инженерную память.

Обязательные элементы памяти этапа:
- Stage Passport
- Builders Log
- Maturity Index
- Recovery Playbook
- Runtime diagnostics
- Restore Tag

Цель: любой будущий инженер должен быстро понять, что было сделано, почему, как проверить и как восстановить состояние.
