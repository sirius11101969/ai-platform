# 03 - Color System

## Base Atmosphere

The main AS6 background is light blue, airy, deep and calm.

It must never become black, heavy or aggressive.

## Core Colors

- calm blue for primary text
- soft white for glass surfaces
- warm gold for finance and growth
- gentle cyan for discovery
- soft violet for care and support
- green only for calm positive presence

## Contrast Rule

Text must remain easy to read.

White text on bright living backgrounds should be avoided unless readability is proven.

## Intelligence Rule

More intelligence must not mean stronger color.

More intelligence should create more calm.
