# 01 - Living Space Design System v1.0

## Master Screen

AS6 Screen 3, Living Space, is the master visual reference for the whole product.

All product areas must be designed as states of this master space, not as unrelated pages.

## Core Rule

AS6 has one brand visual language.

Every screen inherits:

- light blue atmospheric background
- living spheres
- soft glass surfaces
- calm typography
- living relationships
- right-side thinking column
- bottom recommendation block
- conversational input field

## Design Goal

The user should feel that the business space is alive, calm and useful.

## Governance

Any new interface that does not feel like a state of the master screen fails design governance.
