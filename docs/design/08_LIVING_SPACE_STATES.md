# 08 - Living Space States

## Principle

AS6 does not have disconnected screens.

AS6 has states of one Living Space.

## Initial States

- Focus
- Conversation
- Living Space
- Finance
- CRM
- Sales
- Marketing
- Documents
- Team
- Support
- Control

## State Behavior

Each state changes:

- central focus
- active spheres
- right column meaning
- bottom recommendation
- relationship emphasis
- input prompt

Each state preserves:

- background
- spatial system
- typography
- component language
- calm emotional tone

## Rule

A state transition should feel like the space turning attention, not like a page opening.
