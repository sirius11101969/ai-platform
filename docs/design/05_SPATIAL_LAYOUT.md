# 05 - Spatial Layout

## Principle

AS6 uses space as a design material.

The interface should feel open, calm and breathable.

## Master Layout

The master layout comes from Screen 3 Living Space:

- central living object
- surrounding business spheres
- right-side thinking column
- horizontal thinking timeline
- bottom recommendation block
- conversational input field

## Rules

- Keep large empty areas.
- Avoid crowded modules.
- Preserve the center as the strongest focus.
- Keep the right column calm and readable.
- Keep bottom actions stable across states.

## State Rule

A new product area changes the state of the layout, not the layout itself.
