# 07 - Motion System

## Principle

AS6 motion must feel alive, not animated.

## Allowed Motion

- slow breathing glow
- gentle relationship flow
- rare discovery impulse
- smooth state transition
- soft focus shift
- calm appearance of new thoughts

## Forbidden Motion

- blinking
- aggressive pulsing
- fast particles
- gaming effects
- distracting transitions
- motion that competes with reading

## Living Rule

The space should never feel frozen.

The user should also never feel distracted.
