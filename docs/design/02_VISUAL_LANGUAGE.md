# 02 - Visual Language

## Principle

AS6 must look like one calm living business space.

## Visual DNA

- airy blue depth
- warm intelligent glow
- soft translucent glass
- living spheres
- flowing relationships
- human readable text
- no heavy dashboard feeling

## Forbidden

- unrelated page styles
- dark sci-fi screens
- aggressive neon
- dense admin panels
- visual noise

## Recognition Rule

A screenshot should be recognizable as AS6 even without the logo.
