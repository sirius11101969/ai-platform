# 04 - Typography

## Principle

Typography in AS6 must feel calm, clear and premium.

## Rules

- Use calm blue text on light backgrounds.
- Prefer short human phrases.
- Avoid technical labels on user-facing screens.
- Keep hierarchy simple.
- Use generous spacing.
- Make the main focus readable in one glance.

## Forbidden

- dense paragraphs in the main space
- tiny unreadable labels
- white text over bright complex backgrounds
- dashboard-like metric overload

## Voice

The interface should speak like a calm business partner, not like a machine console.
