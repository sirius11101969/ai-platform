# AS6 Living Space Constant Atmosphere Rule

## Status

APPROVED

## Purpose

This rule protects the unified AS6 brand atmosphere across all Living Spaces.

AS6 spaces may have different personalities, but they must never look like different products.

## Core Rule

The base atmosphere of AS6 remains constant across all spaces.

The identity of each space is created by its central composition, objects, connections and accent light — not by replacing the whole background or changing the text language.

## Always Constant Across Spaces

- main soft sky-blue AS6 background;
- overall lightness level;
- calm AS6 blue text color;
- typography;
- logo placement;
- top bar structure;
- right information column structure;
- bottom recommendation card structure;
- input field structure;
- glass transparency language;
- calm premium visual tone.

## Allowed To Change Per Space

- central object silhouette;
- internal geometry;
- object shapes;
- connection logic;
- accent glow;
- emotional tone;
- secondary highlight color;
- right column content;
- bottom card content;
- recommended action.

## Finance Space Correction

For the Finance space:

- restore the same main sky-blue AS6 background used by Focus;
- remove the warm gray-beige background shift;
- keep amber and gold only as accent light around the financial crystal and objects;
- reduce overly saturated blue text;
- use calm AS6 blue-gray text tone consistent with Focus;
- preserve the crystal identity, financial objects and amber financial energy.

## AS6 80/20 Rule

80% of the interface must remain consistent across all spaces:

- atmosphere;
- layout;
- background;
- typography;
- interface structure;
- base text color;
- glass language.

20% creates the identity of the space:

- central composition;
- geometry;
- objects;
- links;
- accent light;
- emotional signature.

## Product Principle

A user should feel that they are still inside AS6, but in another Living Space.

They should never feel that they opened a different website, dashboard or product.
